# Configuration

Create an src/main/resources/application.properties file by copying the provided example files (application.example.properties). In this file, set the following parameters:

* spring.datasource.url - The URL of the SQL Database
* spring.datasource.username - The username to connect to the database
* spring.datasource.password - The corresponding password
